# Daniel Makerv

Portafolio web hecho con tailwind Css

## Construir archivo css con los estilos de Tailwind:

Use el comando:

```bash
npm run build:css
```
## Construir archivo con estilos compilados:

Use el comando:

```bash
npm run build
```
